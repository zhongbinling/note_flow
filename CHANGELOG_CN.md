# 更新日志

本项目的所有重要变更都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)，
本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [0.1.0] - 2026-02-23

### 新增

#### 编辑器
- Markdown 编辑器，支持 4 种模式：编辑、分屏、预览、富文本（所见即所得）
- 实时预览，带代码语法高亮
- 格式工具栏，支持粗体、斜体、标题、列表、引用、代码、链接
- 图片插入，支持粘贴或文件上传，自动压缩（>10KB）
- 自动保存，1000ms 防抖
- 常用格式操作的键盘快捷键

#### 富文本模式
- 使用 contentEditable 实现所见即所得编辑
- 使用 Turndown 进行 HTML 到 Markdown 转换
- 直接插入 HTML 图片元素
- 编辑时保留格式

#### 组织管理
- 文件夹管理（创建、重命名、删除）
- 在文件夹之间移动笔记
- 新笔记默认放入"General"文件夹
- 侧边栏文件夹过滤

#### 搜索
- 全文搜索标题、内容、标签
- 搜索结果高亮
- 实时过滤

#### 用户界面
- 暗色模式，支持系统偏好检测
- 文档标题大纲导航
- 响应式三栏布局
- 保存状态指示器
- 平滑过渡和动画

#### 导入导出
- 导出单个笔记为 Markdown（.md）
- 导出所有笔记为 JSON 备份
- 导入 Markdown 文件
- 导入 JSON 备份

#### 测试
- Playwright E2E 测试配置
- 核心功能测试用例

### 修复
- 大纲点击导致工具栏消失（浏览器布局 bug 与 `scrollIntoView`）
- 预览模式大纲导航位置不准确
- 富文本模式内容同步问题
- 快速切换笔记时的竞态条件
- 富文本模式下格式修改不保存
- 编辑器光标位置问题
- 所有编辑器模式的滚动行为

### 技术实现
- React 19 + TypeScript + Vite 配置
- Tailwind CSS 样式
- Zustand 状态管理，带持久化
- CodeMirror Markdown 编辑
- Lucide React 图标
